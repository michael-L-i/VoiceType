# Taste — VoiceType

> Judgment taught by example. The agent pattern-matches against these.
> **This file grows automatically:** every escalation a human answers gets
> appended here, so taste compounds.

## Good
- **about:** transcription cleanup
  **example:** removing "um/uh" and adding sentence punctuation, while preserving
  the speaker's actual words and intent.
  **why:** we clean up delivery, not rewrite meaning.

- **about:** privacy defaults
  **example:** local model selected on first run; cloud is an explicit toggle with
  a one-line explanation of what leaves the device.
  **why:** the privacy promise must be the path of least resistance.

## Bad
- **about:** onboarding
  **example:** requiring sign-in before the user can dictate a single word.
  **why:** core dictation must work instantly, offline, with no account.

- **about:** configurability
  **example:** shipping five formatting modes behind a settings panel.
  **why:** a setting is a decision we failed to make. Pick the right default.
